# 🏃 CIREBON QRIS RUN 2026 — Sistem Registrasi Event Premium

Sistem registrasi online lengkap untuk acara **CIREBON QRIS RUN 2026** yang merupakan bagian dari **TRIKARSA FEST**. Event ini diselenggarakan oleh **Bank Indonesia Cirebon** dengan dukungan UMKM lokal dan sistem pembayaran QRIS.

## ✨ Fitur Utama

- ✅ **Frontend Premium**: Landing page + multi-step registration form
- ✅ **Backend Robust**: Node.js + Express + MongoDB
- ✅ **Email Automation**: Konfirmasi, verifikasi, dan notifikasi otomatis via Gmail SMTP
- ✅ **QRIS Integration**: Validasi pembayaran otomatis berdasarkan nominal
- ✅ **Cloud Storage**: File upload ke Cloudinary
- ✅ **Admin Dashboard**: Manajemen peserta dan verifikasi real-time
- ✅ **Responsive Design**: Mobile-first, modern glassmorphism UI
- ✅ **Security**: Validasi ketat, duplikasi check, rate limiting

## 🏗️ Tech Stack

**Frontend**: HTML5 / CSS3 / JavaScript (Vanilla)  
**Backend**: Node.js + Express.js + MongoDB  
**Email**: Nodemailer (Gmail SMTP)  
**Storage**: Cloudinary
